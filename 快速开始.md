## 🚀 快速开始（第1周）

### 步骤1：获取 DeepSeek API Key

1. 访问 https://platform.deepseek.com
2. 注册并登录
3. 在控制台创建 API Key
4. 保存好你的 API Key（只显示一次！）

### 步骤2：配置环境

```bash
# 进入第1周目录
cd phase1-api-integration/week1-api-basics

# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入你的 API Key
# DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
```

### 步骤3：运行测试

**方式1：使用 curl（快速测试）**

```bash
# Windows PowerShell
$env:DEEPSEEK_API_KEY="你的密钥"
# 然后查看 curl-examples/README.md 中的示例命令

# Linux/Mac
export DEEPSEEK_API_KEY="你的密钥"
bash curl-examples/test-basic.sh
```

**方式2：使用 Node.js（推荐）**

```bash
cd nodejs-test

# 安装依赖
npm install

# 运行基础测试
npm test

# 运行流式输出测试
npm run test:stream

# 运行多轮对话测试
npm run test:conversation
```

## 💡 学习建议

1. **动手实践最重要**：不要只看代码，一定要自己运行和修改
2. **从简单开始**：先把基础的 API 调用搞明白，再学高级功能
3. **记录问题**：遇到问题记录下来，尝试自己解决，解决不了就搜索或提问
4. **循序渐进**：不要跳过步骤，每周的内容都是为下一周打基础

## 🆘 遇到问题？

### 常见问题

**API Key 无效**
- 检查是否正确复制了完整的 Key
- 检查 .env 文件格式是否正确
- 确保没有多余的空格或引号

**网络连接问题**
- 检查是否能访问 https://api.deepseek.com
- 可能需要配置代理

**依赖安装失败**
- 确保 Node.js 版本 >= 18
- 尝试使用 npm、yarn 或 pnpm
- 检查网络连接

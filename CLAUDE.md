# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a documentation/learning resources directory focused on AI application development. The primary content is a Chinese-language 6-month learning plan (大模型应用开发.md) that outlines a comprehensive path from frontend development to full-stack AI engineering.

## Content Structure

The main document covers:
- **Phase 1 (Months 1-2)**: Quick start with AI integration using OpenAI API and LangChain.js
- **Phase 2 (Months 3-4)**: Core skills including Python, FastAPI, RAG architecture, and vector databases
- **Phase 3 (Month 5)**: Full-stack project development with complex Agent systems
- **Phase 4 (Month 6)**: Advanced topics including LoRA fine-tuning and deployment with Docker

## Technical Stack Referenced

The document references the following technologies:
- **Frontend**: React, Vue, Next.js, Tailwind CSS, Zustand
- **Backend**: Python, FastAPI, LangChain
- **AI/ML**: OpenAI API, RAG architecture, Embeddings, Vector databases (Chroma, Pinecone, Weaviate)
- **Deployment**: Docker, Vercel, Nginx, Cloud platforms (Alibaba Cloud, Tencent Cloud)
- **Tools**: LangSmith, Ollama, Llama 3, Qwen

## Notes

This is a planning/documentation directory, not an active codebase. There are no build commands, test suites, or executable code to run. The content serves as a curriculum guide for aspiring AI application developers.

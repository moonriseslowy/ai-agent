# Week 3 快速开始指南

## 🚀 快速上手（5分钟）

### 1. 进入项目目录
```bash
cd d:\Code\ai-agent\phase1-api-integration\week3-langchain
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置 API Key
创建 `.env` 文件（复制 `.env.example`）：
```
DEEPSEEK_API_KEY=你的API密钥
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
```

### 4. 运行第一个示例
```bash
npm run example1
```

你应该看到 AI 的回复！

## 📚 学习顺序

按照以下顺序运行示例，循序渐进：

```bash
# 示例 1: 基础聊天（5分钟）
npm run example1

# 示例 2: Prompt 模板（10分钟）
npm run example2

# 示例 3: 对话记忆（15分钟）
npm run example3

# 示例 4: 完整对话链（自动演示）
npm run example4:demo

# 示例 4: 交互式对话（亲自体验）
npm run example4
```

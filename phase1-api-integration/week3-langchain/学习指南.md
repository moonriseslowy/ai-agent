## 常见问题解答

### Q1: 运行示例时提示模块找不到？
确保已经安装依赖：
```bash
npm install
```

### Q2: API 调用失败？
检查 `.env` 文件是否正确配置了 API Key。

### Q3: 示例运行很慢？
这是正常的，AI 生成需要时间。如果超过 30 秒，检查网络连接。

### Q4: BufferMemory 和 BufferWindowMemory 的区别？
- BufferMemory: 保存所有历史
- BufferWindowMemory: 只保存最近 k 轮

### Q5: ConversationSummaryMemory 什么时候生成摘要？
每次调用时都会生成摘要，这会产生额外的 API 调用和费用。

### Q6: 如何在 React 中使用这些代码？
Week 4 会详细讲解。主要挑战是 LangChain 默认在 Node.js 环境运行。

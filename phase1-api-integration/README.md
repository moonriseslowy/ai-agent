# 阶段1：快速上手与"AI感"建立

## 学习目标
- 掌握 DeepSeek API 的基本调用
- 理解大模型的请求和响应格式
- 实现简单的对话功能
- 为后续 LangChain.js 学习打基础

## 周度安排

### 第1周：API 基础测试
**目标**：注册 DeepSeek 账号，获取 API Key，用 curl 和 Node.js 测试基本调用

**学习内容**：
- DeepSeek API 文档阅读
- 使用 curl 测试 API
- 使用 Node.js SDK 测试 API
- 理解 chat completions 接口

### 第2周：简单 Web Demo
**目标**：创建一个极简的网页对话 demo

**学习内容**：
- 使用 Vite 或 Next.js 创建项目
- 集成 DeepSeek API
- 实现基本的对话界面

### 第3周：LangChain.js 入门
**目标**：学习 LangChain.js，实现带记忆的对话

**学习内容**：
- LangChain.js 核心概念
- ChatPromptTemplate 使用
- ConversationSummaryMemory 实现

### 第4周：功能增强
**目标**：实现文档问答或联网搜索

**学习内容**：
- 文本分割和处理
- 简单的检索问答
- 工具集成（可选）

## DeepSeek API 信息

- **API 地址**: https://api.deepseek.com
- **文档地址**: https://platform.deepseek.com/api-docs
- **兼容性**: DeepSeek API 兼容 OpenAI SDK，可以直接使用 OpenAI 的 Node.js SDK

## 快速开始

1. 注册 DeepSeek 账号：https://platform.deepseek.com
2. 获取 API Key
3. 进入 `week1-api-basics` 目录开始学习
